# When running `leanblueprint web`

Leanblueprint may complain about not having permissions to edit a few files.

Those files are listed below for convenience.

```
chmod +rw blueprint/web/js/svgxuse.js blueprint/web/js/jquery.min.js blueprint/web/js/plastex.js blueprint/web/styles/* blueprint/web/symbol-defs.svg
```
